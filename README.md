# 🔫 狂暴老奶 - Grandma Rampage 🔫

一个微信小游戏 - 像素风格的Roguelike游戏

## 游戏特性

✨ **像素艺术风格** - 复古8位像素美学

🎮 **Roguelike机制** - 随机关卡，永久升级

👵 **老奶奶主角** - 可升级的武器系统

👹 **波次敌人** - 难度逐渐上升

💪 **升级系统** - 攻击力、射速、生命值、射程

🎯 **多关卡** - 无限波次挑战

## 游戏玩法

1. **移动**: 点击屏幕移动老奶奶到目标位置
2. **攻击**: 自动射击范围内的怪兽
3. **升级**: 击杀怪兽获得升级点数
4. **生存**: 存活尽可能多的波次

## 核心游戏类

### Game
- 主游戏循环
- 敌人波次管理
- 碰撞检测
- 粒子效果

### Player
- 玩家移动
- 自动瞄准和射击
- 升级系统
- 生命值管理

### Enemy
- 敌人AI (朝玩家移动)
- 两种敌人类型 (小怪/大怪)
- 生命值系统

### Projectile
- 投射物物理
- 碰撞检测
- 伤害系统

### Particle
- 粒子效果
- 重力模拟

## 升级系统

| 升级项 | 效果 | 花费 |
|--------|------|------|
| 🔴 攻击力 | +1伤害 | 1点 |
| ⚡ 射速 | +1射速等级 | 1点 |
| 💚 生命值 | +5生命值 | 1点 |
| 🎯 射程 | +50px攻击范围 | 2点 |

## 敌人类型

### 小怪 (Goblin) - 绿色
- 血量: 20
- 速度: 标准
- 分布: 混合

### 大怪 (Orc) - 紫色
- 血量: 20
- 速度: 标准
- 分布: 混合

## 游戏难度

- **波次1**: 3个敌人
- **波次N**: 3 + floor(N/2) 个敌人
- 敌人每波逐渐增加
- 距离玩家越近敌人越快

## 开发与部署

1. 使用微信开发者工具打开此项目
2. 在 game.json 中配置小游戏信息
3. 进行预览和测试
4. 提交审核发布

## 游戏流程

```
主菜单
  ↓
游戏说明 (可选)
  ↓
开始游戏
  ↓
击杀敌人 → 获得升级点 → 升级选择 → 继续
  ↓
生命值为0 → 游戏结束
  ↓
查看统计 → 重新开始/返回菜单
```
