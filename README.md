# 🔫 狂暴老奶 - Grandma Rampage 🔫

一个完整的微信小游戏 - 像素风格的Roguelike游戏，无限波次挑战！

## ✨ 游戏特性

### 核心玩法
- 🎮 **像素艺术风格** - 复古8位像素美学
- 🌊 **Roguelike机制** - 随机关卡，永久升级
- 👵 **老奶奶主角** - 可升级的武器和属性系统
- 🌊 **无限波次** - 波次越高难度越大
- 💪 **升级系统** - 攻击力、射速、生命值、射程、速度

### 武器系统
- 🔫 **普通枪** - 基础射击
- 🔥 **喷火枪** - 发射多枚子弹
- ⚡ **激光枪** - 高穿透力伤害
- 💥 **火箭枪** - 爆炸伤害，高伤害低射速

### 敌人类型
- 👺 **小怪 (Goblin)** - 绿色，速度快
- 👿 **兽人 (Orc)** - 紫色，血量多
- 💀 **骷髅 (Skeleton)** - 灰色，均衡型
- 😈 **恶魔 (Demon)** - 红色，强大敌人
- 👹 **Boss** - 每5波一次，超强敌人

### 掉落物品
- 💚 **生命药水** - 恢复20点生命值
- 💰 **金币** - 获得10点金币

### 难度系统
- 😊 **简单** - 敌人较弱，升级更快，非常适合新手
- 😐 **普通** - 平衡的挑战，推荐难度
- 😡 **困难** - 强大的敌人和BOSS，有经验的玩家
- 💀 **地狱** - 终极挑战，仅供高手！

### 成就系统 🌟
- 🎮 **初学者** - 完成第1波
- 🌊 **冲浪者** - 完成第10波
- 🏄 **浪人** - 完成第25波
- 👑 **波次之王** - 完成第50波
- 💀 **猎手** - 击杀100个敌人
- 🔥 **屠夫** - 击杀500个敌人
- 😡 **勇者** - 在困难模式下完成第10波
- 💀 **疯子** - 在地狱模式下完成第5波
- 🔥 **火焰战士** - 解锁喷火枪
- ⚡ **激光侠** - 解锁激光枪
- 👹 **Boss猎人** - 击败第一个Boss
- 🎯 **武器大师** - 解锁所有武器

### 排行榜系统
- 🏆 **本地排行榜** - 保存你的最佳成绩
- 📊 **自动排序** - 按分数自动排序
- 👥 **竞争** - 与自己竞争，刷新纪录

## 🎮 游戏玩法

### 基本操作
1. **移动** - 点击屏幕任意位置移动老奶奶
2. **攻击** - 自动射击范围内的敌人
3. **升级** - 完成每波后选择升级
4. **收集** - 击杀敌人掉落的物品会自动收集

### 升级策略
- 💪 **属性升级** - 永久提升玩家属性
  - 🔴 攻击力 - 提高伤害输出
  - ⚡ 射速 - 提升射击频率
  - 💚 生命值 - 增加总血量
  - 🎯 射程 - 扩大攻击范围
  - 💨 速度 - 提升移动速度

- 🔫 **武器升级** - 解锁强力武器
  - 🔥 喷火枪 - 一次发射3枚子弹
  - ⚡ 激光枪 - 穿透力强，伤害高
  - 💥 火箭枪 - 爆炸伤害，超强输出

## 📊 游戏平衡

### 波次难度
- **波次1-4** - 初级敌人，容易生存
- **波次5** - 第一个Boss，中等难度
- **波次6-9** - 多种敌人混合
- **波次10** - 第二个Boss，高难度
- **波次11+** - 难度快速上升

### 敌人数量
- 基础 = 3 + floor(波次/2)
- 难度系数 = {简单:0.8, 普通:1, 困难:1.2, 地狱:1.5}

### 伤害计算
- 基础伤害 = 10
- 升级后 = 基础 + (升级次数 × 2)
- 武器倍数 = {普通:1, 喷火:0.8×3, 激光:1.5, 火箭:2}

## 🚀 快速开始

### 安装和运行
1. 下载或克隆此项目
2. 打开微信开发者工具
3. 导入项目文件夹 `111`
4. 在 `project.config.json` 中填入你的AppID
5. 点击"预览"或"编译"测试游戏

1. 使用微信开发者工具的"预览"功能

## 🌟 高级技巧

### 武器选择
- **普通枪** - 平衡选择，升级速度快
- **喷火枪** - 群伤输出，对付多敌人
- **激光枪** - 单体伤害，适合Boss战
- **火箭枪** - 超高伤害，但射速慢

### 属性优先级
1. 早期 - 优先升级生命值和攻击力
2. 中期 - 平衡升级射速和射程
3. 后期 - 根据难度选择防守或输出

### 走位技巧
- 不要站在敌人密集处
- 利用屏幕边缘躲避
- 集中火力消灭单个敌人
- Boss战时保持距离
